# MountTracker

A WIP Dalamud plugin for Final Fantasy XIV that tracks mount acquisition status for selected players.

## Overview

MountTracker is a plugin designed for Dalamud, through XIVLauncher. This plugin allows you to monitor the mount collection progress of selected players. Many mounts in FFXIV are obtained by defeating specific bosses or completing challenging content, making it useful to track which mounts you and your friends have collected.

## Features

* **Player Selection**: Choose the players you want to track in the configuration window
    * **(WIP) Context Menu**: Add players from the party list through the context menu
* **Mount Selection**: Select specific mounts you want to monitor
* **Main Display**: View mount acquisition status for all tracked players in an easy-to-read interface
* **(WIP) Real-time Updates**: Monitor progress as players acquire new mounts

## Installation

```
https://raw.githubusercontent.com/Cyenia/DalamudPlugins/master/pluginmaster.json
```

* Copy the link above
* Open your Plugin Installer Window and click Settings
* Go to the Experimental Tab
* Paste the link into the Custom Plugin Repositories, and click the + on the right to add it.
* Save and close.
* Search and Install MountTracker from the Plugin Installer.
* Enjoy

## Usage

1. **Opening the Plugin**: Use `/mounttracker` in chat or through the Dalamud plugin menu
2. **Configuration**:
    - Access the settings to add players you want to track
    - Select which mounts you're interested in monitoring
    - Configure display preferences
3. **Monitoring**: The main window will show the acquisition status of selected mounts for all tracked players

## Commands

* `/pmount` - Opens the main MountTracker window
